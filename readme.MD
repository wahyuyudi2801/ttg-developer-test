# Intruksi untuk menjalankan program
## Soal 1
Buka file atau copy path index.html di browser (chrome, firefox, dll)

## Soal 2
jalankan perintah
```
npm install
```
lalu 
```
npm run dev
```
Maka server akan berjalan di port 3000.
Anda sudah bisa melakukan pendaftaran melalui API (Gunakan postman/insomnia).
