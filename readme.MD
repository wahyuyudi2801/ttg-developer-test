# Intruksi untuk menjalankan program
## Soal 1
1. masuk ke folder soal-1
2. Buka file atau copy path index.html di browser (chrome, firefox, dll)

## Soal 2
Requirement: NodeJS, Apache/Nginx, MySQL

### Setup database
1. Nyalakan server Apache/Nginx dan MySQL
2. Buka phpmyadmin
3. Import database yang ada di direktori soal-2 (db_ttg_test.sql)

### Jalankan program
1. Buka terminal.
2. Masuk ke direktory soal-2
```
cd soal-2
```
3. jalankan perintah
```
npm install
```
4. Jalankan server 
```
npm run dev
```
Maka server akan berjalan di port 3000.
Anda sudah bisa melakukan pendaftaran melalui API (Gunakan postman/insomnia).
